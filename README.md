# 🐍 Змейка на Python (Pygame)

Простая реализация классической игры «Змейка» с использованием библиотеки `pygame`.

## 📌 Описание

В этой игре ты управляешь змейкой, собираешь яблоки и увеличиваешь свой счёт. При столкновении змейки с самой собой игра начинается заново.

## 🎮 Управление

* ⬆️ Стрелка вверх — движение вверх
* ⬇️ Стрелка вниз — движение вниз
* ⬅️ Стрелка влево — движение влево
* ➡️ Стрелка вправо — движение вправо

## ⚙️ Функционал

* Рост змейки при поедании яблок
* Подсчёт очков
* Генерация яблок в случайных местах
* Перезапуск игры при столкновении с собой
* Переход через границы экрана (змейка появляется с другой стороны)

## 🚀 Установка

1. Установи Python (если ещё не установлен)
2. Установи библиотеку `pygame`:

```bash id="inst1"
pip install pygame
```

## ▶️ Запуск

```bash id="run1"
python zmayka.py
```

## 🛠️ Настройки

Ты можешь изменить параметры игры в коде:

* Размер окна:

```python id="cfg1"
WIDTH = 800
HEIGHT = 600
```

* Размер клетки:

```python id="cfg2"
cell_size = 20
```

* Скорость игры:

```python id="cfg3"
clock.tick(10)
```

(Чем больше значение, тем быстрее змейка)

## 🧠 Особенности реализации

* Змейка хранится как список координат
* Новая голова добавляется в начало списка
* При движении хвост удаляется (если не съедено яблоко)
* Используется зацикливание координат через `%`, чтобы змейка проходила через стены
